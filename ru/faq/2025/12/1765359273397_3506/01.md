- Создать файл `/etc/postfix/header_checks_smtp` со следующим содержанием:

```
/^X-Mailer:/ IGNORE
/^X-Mailer-Type:/ IGNORE
/^User-Agent:/ IGNORE
/^Received:/ IGNORE
/^X-Originating-IP:/ IGNORE
/^X-PHP-Originating-Script:/ IGNORE
```

- Создать карту файла `header_checks_smtp`:

```bash
postmap '/etc/postfix/header_checks_smtp'
```

- Открыть файл `/etc/postfix/main.cf`, добавить:

```ini
smtp_header_checks = pcre:/etc/postfix/header_checks_smtp
smtp_mime_header_checks = pcre:/etc/postfix/header_checks_smtp
```

- Перезапустить `postfix`:

```bash
systemctl restart 'postfix.service'
```
