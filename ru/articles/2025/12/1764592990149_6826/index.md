## Скрипт

[Скрипт](https://github.com/pkgstore/pwsh-mail) позволяет отправлять {{< tag "email" >}} через терминал {{< tag "PowerShell" >}}.

### Установка

### Настройка

- `Server` - IP-адрес сервера.
- `Port` - SMTP-порт сервера.
- `User` - имя пользователя для SMTP-аутентификации.
- `Password` - пароль пользователя для SMTP-аутентификации.

### Параметры

- `Subject` - заголовок письма.
- `Body` - содержание письма.
- `From` - поле "От".
- `To` - поле "Кому".
- `Cc` - поле "Копия".
- `Bcc` - поле "Скрытая копия".
- `Attachment` - путь к файлу или файлам для вложения в письмо.
- `Priority` - приоритет письма.
- `HTML` - письмо будет иметь разметку HTML.
- `SSL` - отправка письма через зашифрованный протокол SSL.
- `BypassCertValid` - отключить проверку валидности сертификата при отправки письма через протокол SSL.

## Примеры

```terminal {os="windows",lang="pwsh"}
.\app.mail.ps1 -Subject 'Example' -Body 'Hello world!' -From 'mail@example.com' -To 'mail@example.org'
```

```terminal {os="windows",lang="pwsh"}
.\app.mail.ps1 -Subject 'Example' -Body 'Hello world!' -From 'mail@example.com' -To 'mail@example.org' -Attachment 'C:\file.01.txt', 'C:\file.02.txt'
```
