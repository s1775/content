Команды для "правильной" записи {{< tag "ISO" >}} на {{< tag "USB" >}} Flash Drive.

<!--more-->

## План работы

- Определить {{< tag "USB" >}}-накопитель при помощи команды `fdisk -l`.
- Отмонтировать ВСЕ разделы с {{< tag "USB" >}}-накопителя командой `umount /dev/sdX*`, где `X` - буква накопителя (например, `umount /dev/sdc*`).
- Загрузить {{< tag "ISO" >}} на {{< tag "USB" >}}-накопитель.

## Экспорт переменных

- Экспортировать заранее подготовленные общие параметры в переменные окружения:

```bash
export SD_X='/dev/sdX'
```

### Параметры

- `/dev/sdX` - наименование USB Flash Drive в системе.

## Запись

- От-монтировать и очистить USB-накопитель:

```bash
umount ${SD_X}* && sgdisk -Z ${SD_X}
```

- Записать образ на USB-накопитель:

```bash
dd if=image.iso of=${SD_X} bs=4M oflag=direct status=progress; sync
```

- Извлечь USB-накопитель:

```bash
eject ${SD_X}
```
